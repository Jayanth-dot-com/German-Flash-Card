# German-Flash-Card
Beginner level German learning Flash card
# Key Features
1. This flash card contains 1400+ comman german words in slides
2. Each slide contains:
3.   *German word/phrase
4.   *English meaning/translation
5. Covers everyday language - nouns, verbs, adjectives, and common expressions too
# How to Use
1. Download the deck file from this repository
2. Open the file using your text editor(VS Code, Pycharm, etc..)
3. Then update the file location of the images
4. Now the program is ready to use
